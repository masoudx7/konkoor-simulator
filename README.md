# 🎓 شبیه‌ساز کنکور سراسری

یک اپلیکیشن آموزشی برای شبیه‌سازی کنکور سراسری با رابط کاربری گرافیکی ساده و کارآمد.

## ✨ ویژگی‌های پروژه

- 🎯 **سوالات متنوع**: سوالات از درس‌های مختلف (ریاضی، فیزیک، شیمی، زیست‌شناسی، ادبیات)
- 📊 **نمره‌دهی خودکار**: محاسبه خودکار نمره و درصد نتایج
- 🏆 **رتبه‌بندی**: نمایش رتبه‌بندی بر اساس درصد
- 📋 **تاریخچه**: ثبت و مشاهده تمام کوییزهای انجام شده
- 🎨 **رابط دوستانه**: طراحی ساده و استفاده آسان
- 🔄 **تنوع سوالات**: انتخاب تصادفی سوالات در هر کوییز

## 🚀 نحوه نصب و اجرا

### الزامات سیستم:
- Python 3.6+
- Tkinter (معمولاً با Python نصب می‌شود)

### مراحل نصب:

#### 1. کلون کردن مخزن:
```bash
git clone https://github.com/masoudx7/konkoor-simulator.git
cd konkoor-simulator
```

#### 2. نصب وابستگی‌ها (اختیاری):
```bash
pip install -r requirements.txt
```

#### 3. اجرای برنامه:
```bash
python main.py
```

یا در Linux/Mac:
```bash
python3 main.py
```

## 📖 راهنمای استفاده

### صفحه اصلی:
1. برنامه را اجرا کنید
2. یکی از درس‌ها را انتخاب کنید (ریاضی، فیزیک، شیمی، زیست‌شناسی، یا ادبیات)
3. تعداد سوالات مورد نظر را وارد کنید (۱ تا ۱۰ سوال)

### حل کوییز:
1. هر سوال را به‌دقت بخوانید
2. یکی از چهار گزینه را انتخاب کنید
3. دکمه "→ بعدی" را کلیک کنید
4. این کار را برای تمام سوالات تکرار کنید

### نمایش نتایج:
- پس از پایان کوییز، نتایج نمایش داده می‌شود
- می‌توانید امتیاز، درصد و رتبه‌بندی خود را مشاهده کنید
- برای دیدن تفاصیل دقیق پاسخ‌ها، دکمه "📋 مشاهده تفاصیل" را کلیک کنید

### مشاهده تاریخچه:
- از صفحه اصلی، دکمه "📊 مشاهده تاریخچه" را کلیک کنید
- تمام کوییزهای قبلی و نتایج آنها را مشاهده کنید

## 🛠️ توسعه �� اضافه کردن سوالات

### اضافه کردن سوالات جدید:

فایل `questions.py` را باز کنید و سوالات جدید را اضافه کنید:

```python
QUESTIONS = {
    "درس_جدید": [
        {
            "سوال": "متن سوال",
            "گزینه‌ها": ["گزینه 1", "گزینه 2", "گزینه 3", "گزینه 4"],
            "پاسخ_صحیح": 0  # شماره گزینه صحیح (از ۰ شروع می‌شود)
        },
    ]
}
```

### تغییر طراحی:

رنگ‌ها و فونت‌های استفاده شده در فایل `main.py` را می‌توانید تغییر دهید:

```python
self.فونت_عنوان = ("Arial", 20, "bold")  # فونت عنوان
# ...
bg="#1e3a8a"  # رنگ پس‌زمینه
fg="white"    # رنگ متن
```

## 📊 سیستم نمره‌دهی

نمره حرفی بر اساس درصد:
- **90% و بیشتر**: ممتاز ⭐⭐⭐⭐⭐
- **80% تا 90%**: خیلی خوب ⭐⭐⭐⭐
- **70% تا 80%**: خوب ⭐⭐⭐
- **60% تا 70%**: متوسط ⭐⭐
- **50% تا 60%**: قابل قبول ⭐
- **کمتر از 50%**: ضعیف
